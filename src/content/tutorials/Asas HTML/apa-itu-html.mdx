HTML atau (Hypertext Markup Language) adalah asas kepada semua website. Anda boleh anggap HTML sebagai batu-bata dan website sebagai bangunan.

Untuk pengetahuan anda, HTML tidak dikategorikan sebagai _programming language_, tetapi merupakan sebuah _markup language_. Anda tak perlu faham perbezaan antara kedua-dua _language_ ni untuk permulaan.

_Markup Language_ seperti HTML menggunakan **tag** sebagai peraturan bahasa atau _sintaks_.

contoh tag yang anda akan selalu nampak adalah seperti `<img>`, `<p>`, `<main>`, `<section>` dan banyak lagi.

Perasan tak? setiap tag yang ditunjukkan tadi mempunyai nama mereka sendiri.

Selain menjadi asas kepada pembinaan website, HTML juga digunakan untuk menentukan struktur sesuatu website. Dalam satu website akan ada _item_ seperti teks, gambar, _link_, dan _button_. Selain itu, setiap website juga mempuyai layout yang tersendiri dengan bahagian seperti _header_, _content_, _sidebar_ dan _footer_.

Semua item dan bahagian dalam suatu website dikenali sebagai elemen. Senang cerita, semua benda yang ada dalam sesuatu website tu merupakan elemen.

Setiap elemen diwakili oleh tag HTML yang berbeza. Contohnya:

- Elemen gambar menggunakan tag `<img>`
- Elemen teks menggunakan tag `<p>`
- Elemen _header_ menggunakan tag `<header>`

Terdapat 2 jenis tag iaitu iaitu tag mempunyai _closing tag_ dan yang tidak mempunyai _closing tag_ atau lebih dikenali sebagai _self-closing tag_

Jenis tag pertama mempunyai _opening_ dan _closing_ tag.

```html
<p>Welcome to The TEX Project!</p>
```

Jenis tag kedua tiada _closing tag._

```html
<img src="/image/myimage.png" />
```

Dalam pelajaran seterusnya anda akan belajar cara untuk _coding_ program HTML anda sendiri.

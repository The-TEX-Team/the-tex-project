## Buat fail HTML

Program HTML ditulis di dalam fail berfomat `.html`. Jadi, anda hanya perlu buat fail dengan format `.html` contohnya `index.html`

Kalau anda buka fail tersebut, _web browser_ anda akan terbuka dan paparkan _page_ kosong.

Kalau anda buka fail html secara terus, _browser_ anda akan memaparkan website berdasarkan fail html anda. Sebab fail anda masih kosong, _browser_ hanya akan memaparkan _page_ kosong tanpa sebarang elemen.

> Macam mana nak coding HTML kalau guna browser?

Macam ni, kita buka fail HTML menggunakan browser kalau nak tengok rupa website yang telah diprogram. Untuk anda _coding_ HTML di dalam fail tersebut, anda memerlukan satu program yang dinamakan code editor.

Code editor ni sejenis program yang boleh digunakan untuk _edit_ fail, tak semestinya untuk tujuan _coding._ Program seperti Notepad pun boleh digunakan untuk _coding._ Tapi, code editor dilengakapi dengan pelbagai fungsi yang dikhaskan untuk keperluan semasa _coding_ seperti _syntax highlighting_ dan _code suggestion._

Disebabkan tutorial ini hanya memfokuskan asas HTML, kita akan gunakan program Notepad. _Right Click_ pada fail HTML anda dan click _open with Notepad_

## Struktur fail HTML

```html
<html>
  <head>
    <title>Document</title>
  </head>
  <body>
    <h1>Welcome to The TEX Project</h1>
  </body>
</html>
```

Program di atas merupakan struktur fail HTML yang paling asas bagi suatu website. Anda boleh lihat ada banyak tag yang digunakan untuk menentukan struktur website tersebut.

Kita mulakan dengan tag yang pertama iaitu `<html>`. Tag ini merupakan `root` bagi program HTML ini. Semua tag yang akan dimasukkan ke dalam fail ini mesti diletakkan di dalam tag `<html>`.

Tag `<head>` merupakan tag yang unik. Semua tag yang diletakkan di dalam tag ini tidak akan dipaparkan sebagai elemen pada websitem, tetapi akan digunakan sebagai data atau informasi untuk website seperti tajuk dan _metadata_.

Jika anda buka fail HTML yang mengandungi program di atas menggunakan _browser_, anda akan lihat yang website tersebut sudah ada tajuk pada _tab browser_ anda. Tag `<title>` menentukan tajuk kepada website anda.

Tag bagi semua elemen yang akan dipaparkan pada website seperti teks, gambar atau _link_ perlu diletakkan di dalam tag `<body>`.
